# CMOS NOR Gate Design using Cadence Virtuoso

## Overview
This project demonstrates the design and simulation of a 2-input CMOS NOR gate using Cadence Virtuoso.

## Features
- CMOS NOR gate schematic
- Functional simulation
- Input-output waveform verification
- Digital VLSI design practice

## Tools Used
- Cadence Virtuoso
- Spectre Simulator
